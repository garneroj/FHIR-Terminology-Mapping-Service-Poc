A **Proof of Concept (POC)** for a Clinical Terminology Server designed to map proprietary legacy codes to LOINC standards using the **HL7 FHIR** `$translate` operation.

This project leverages the **HAPI FHIR JPA Starter** to provide a robust, standard-compliant server backed by **PostgreSQL**.

toc::[]

== Introduction

This repository contains a functional implementation of a Terminology Server capable of ingesting CodeSystems and ConceptMaps to perform runtime translations. It is designed to be easily deployed using Docker containers for the persistence layer.
